# Changelog

All notable changes to PerpetualCC will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure
- CLI scaffold with `pcc` command
- Claude Agent SDK adapter for session management
- Event type system for Claude Code messages
- `pcc start` command to run Claude Code sessions
- `pcc version` command
- Rich terminal output formatting
- Streaming event display with timestamps

### Technical
- Python 3.11+ requirement
- Typer-based CLI framework
- Async event streaming architecture
- Comprehensive release documentation

## [0.1.0] - TBD

### Added
- First public release
- Core CLI functionality
- Claude Code session orchestration
- Basic event handling and display
